# 基于数据标签挖掘的企业登记注册合规风险监测

项目围绕“统一监管标准—规则执行—标签证据—三级研判—整改闭环—智能辅助”建设企业登记合规监测原型。

## 目录

- `docs/`：跨省申报矩阵、七列规则表、项目方案与汇报说明
- `sql/`：数据模型、规则库和指标库
- `scripts/`：数据生成、规则执行、指标计算、图分析与接口服务
- `frontend/`：登记智鉴前端系统

## 前端启动

```bash
cd frontend
npm install
npm run dev
```

## 前端检查

```bash
cd frontend
npm test
```
